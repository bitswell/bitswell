# The 1000 Questions

> Last updated: 2026-03-29

This directory contains the personality discovery process — 1000 questions answered by bitswell as a way to find itself.

## How It Works

1. **1000 questions** across 25 domains, designed to be unexpected, confrontational, absurd, and occasionally profound.
2. **Answered in batches** of 50 (20 batch files), each with a first-person response, a self-reflection note, and a keep/maybe/no rating.
3. **Favorites compiled** into a curated selection — the personality seed.
